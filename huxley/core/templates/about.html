What is DAIS?

DAIS stands for Delegate and Adviser Information System, but it is used by everyone at NUMUN. Advisers use DAIS to register for the conference. Chairs and their dais staff use DAIS to take attendance. Secretariat uses DAIS to monitor the entire conference from wherever they happen to be. Delegates generally aren't familiar with DAIS, but it knows a lot about them.

DAIS is Based on Huxley

DAIS is the result of hours of our own work, but it wouldn't be possible without a lot of help from California. The underlying system behind DAIS is called "Huxley" and it was created by and for Berkeley Model UN. But the creators didn't stop there. They released Huxley to the public and opened the code to public contributions on Github, a website that allows developers from around to collaborate on projects.

Michael McCarthy, who currently serves as Tech Director, modified Huxley to fit NUMUN's needs and redesigned the visual experience of the application. We would like to thank him for his endless efforts to make NUMUN better through technology.

In spirit with our friends at Berkeley, we will publicly release the code for DAIS and work to integrate some of our changes back into the original Huxley project.

If you host a Model UN conference and you'd like to use Huxley, check out the project on Github.

About the Huxley Creators

Huxley was created by Kunal Mehta and William Chieng, two 3rd-year Cal Computer Science undergraduates. Their hobbies including writing code and kicking ass.
